# 个人简历网站 · 设计文档

**日期**：2026-06-19
**作者**：林晖鹏 (Huipeng Lin) · GitHub [@Absurdaaa](https://github.com/Absurdaaa)
**项目**：个人简历 / 主页网站，托管于 GitHub Pages

---

## 1. 目标

为林晖鹏制作一个**个人简历 / 主页网站**（personal portfolio），定位为"展示我是谁"的个人品牌站，而非保研申请材料。

- 风格：**Claude Code 设计感**——奶油纸感背景、衬线标题、等宽字副标题、珊瑚色点缀，克制而有"文气"的极客风。
- 重点：用**科研、论文、竞赛获奖**说话；**不展示 GPA 与排名**。
- 加入**像 GitHub 主页那样的自我介绍 + 兴趣**，体现个性。
- 提供**主题切换开关**，可在 4 套配色间切换。

## 2. 非目标 (YAGNI)

- ❌ 不做后端、CMS、数据库——纯静态站。
- ❌ 不展示 GPA、年级排名、课程成绩。
- ❌ 不做多页路由——单页向下滚动即可。
- ❌ 首版不做博客、留言、多语言切换（中文为主，学术术语保留英文）。

## 3. 技术方案

**纯静态：HTML + CSS + 原生 JS，无构建步骤。**

- 理由：单页简历站、需长期由本人手改内容、直接部署 GitHub Pages，无需框架/打包。
- **配色用 CSS 自定义属性（CSS variables）**：在 `:root` 与 `[data-theme="..."]` 下定义颜色变量，主题切换只需改 `<html>` 的 `data-theme`，用 `localStorage` 记住选择。这让 4 套主题切换非常干净。
- 字体：衬线标题用系统衬线 / Google Fonts（如 Tiempos 替代品 `Noto Serif SC` + 英文 `Newsreader`/`Georgia`）；等宽用 `ui-monospace` / `JetBrains Mono`。中文正文用系统无衬线。
- 响应式：移动端单列堆叠。

**文件结构：**
```
github-profile/
├── index.html
├── css/styles.css
├── js/theme.js          # 主题切换 + localStorage
├── js/main.js           # 滚动/交互（如有）
├── assets/              # 照片、favicon、图标
└── data/ (可选)         # 若把内容抽成 JSON 便于维护
```

部署：推到 `Absurdaaa/Absurdaaa.github.io`（或任意仓库开 Pages），访问 `absurdaaa.github.io`。

## 4. 配色主题（4 套，CSS variables 驱动）

每套定义：`--bg`（页面底）、`--surface`（卡片底）、`--text`、`--muted`、`--accent`。

| 主题 | 名称 | --bg | --surface | --accent | --text |
|------|------|------|-----------|----------|--------|
| A（默认）| 奶油+珊瑚 | `#F0EEE6` | `#FAF9F5` | `#D97757` | `#1F1E1D` |
| B | 奶油+靛蓝 | `#F2F0EA` | `#FBFAF6` | `#5B5BD6` | `#1E1E24` |
| C | 奶油+森绿 | `#EFEEE7` | `#FAF9F4` | `#3F7D5A` | `#1C1E1B` |
| D | 暖调深色 | `#1A1714` | `#24201B` | `#D97757` | `#ECE7DC` |

切换控件：导航右上角四个圆点，颜色对应四主题，点击切换，状态存 `localStorage`。
